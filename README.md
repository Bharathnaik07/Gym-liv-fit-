# Liv.fit - Antigravity Fitness Studio 🤸‍♀️💪

Liv.fit is a modern, responsive, and beautifully designed landing page for a women-focused fitness studio located in Sirsi, Karnataka. The studio provides a supportive and energetic environment to help women build strength, energy, and confidence.

## 🌟 Key Features

- **Modern & Premium Design**: Features a dark-themed UI with glassmorphism (`glass-card`) effects and vibrant highlights.
- **Fully Responsive**: Optimized for desktops, tablets, and mobile devices.
- **Interactive Elements**:
  - Smooth scrolling navigation and sticky header.
  - Testimonial carousel for member reviews.
  - Interactive **BMI Calculator** to check fitness levels instantly.
  - Lightbox gallery for viewing studio photos.
- **Service Highlights**: Dedicated sections showcasing Antigravity Fitness, Zumba/Dance, CrossFit, Aerobics, and more.
- **Quick Contact**: Floating WhatsApp button and Google Maps integration for easy studio location and communication.

## 🛠️ Tech Stack

- **HTML5**: Semantic and structured content.
- **CSS3**: Custom styling, CSS Grid/Flexbox layouts, scroll animations, and glassmorphic UI elements.
- **JavaScript (Vanilla)**: For DOM manipulation, mobile hamburger menu, smooth scrolling, BMI calculation, testimonial carousel, and gallery lightbox.
- **Google Fonts**: Utilizes 'Inter' and 'Outfit' for clean, modern typography.
- **FontAwesome**: Scalable vector icons used throughout the site.

## 🚀 How to Run Locally

1. Clone this repository or download the source code.
2. Navigate to the project directory.
3. Open the `index.html` file in your preferred web browser. 
   - *Tip: Use an extension like **Live Server** in VS Code for a better development experience with live reloading.*

## 📂 Project Structure

- `index.html` - The main structure of the landing page.
- `style.css` - Contains all design tokens, animations, and responsive media queries.
- `script.js` - Contains the logic for interactive elements like the mobile menu, BMI calculator, slider, and lightbox.
